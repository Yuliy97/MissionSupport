This is the directory for templates released by Microsoft for ASP.NET Web Pages.
To use these templates, copy out the contents of the corresponding folder into 
the root of a new website, or, if you started from the Empty Site template, 
into the root of your current site.

Example: To use the Wishlist template starting from the Empty Site template, copy the 
contents of Microsoft Templates\Wishlist into the root of your site.
